# Clip Candidate Scorecard

Use this scorecard to decide which parts of a YouTube video should become Shorts.

## Candidate table

| Timestamp | Moment | Hook strength | Standalone context | Visual interest | Platform fit | Notes |
|---|---|---:|---:|---:|---:|---|
|  |  | 1-5 | 1-5 | 1-5 | 1-5 |  |

## Scoring guide

| Score | Meaning |
|---:|---|
| 1 | Weak, needs too much context |
| 3 | Usable with rewrite or captions |
| 5 | Strong candidate, easy to understand and edit |

## Selection rules

- Prefer moments with a clear question or payoff.
- Avoid clips that need too much earlier context.
- Choose visually active moments when possible.
- Use captions to repair small context gaps.
- If two moments say the same thing, keep the shorter one.
