# YouTube 长视频转 Shorts 剪辑工具包

YouTube to Shorts 剪辑，是把长视频变成竖屏、短时长、可发布的短视频。本仓库整理片段筛选、开头改写、字幕、配音和多平台适配模板，适合把 YouTube 长视频转成 Shorts、Reels、TikTok 和社交短片。

目标是在正式剪辑前，把长视频复用计划先拆清楚。

## 这个工具包覆盖什么

- YouTube 章节到 Shorts 的规划
- 候选片段评分
- 面向竖屏短视频的开头改写
- 字幕和配音备注
- Shorts 成片检查清单
- 多平台适配说明

## 复用格式

| 格式 | 适合素材 | 输出 |
|---|---|---|
| 核心观点切片 | 教程、访谈、Webinar | 30-60 秒讲清一个观点 |
| 故事高光 | Vlog、纪录片、解说 | 一个完整的小故事瞬间 |
| 问题-答案切片 | 产品教育、how-to 视频 | 痛点和答案 |
| 反应/评论切片 | 评论、娱乐内容 | 强观点或强反应 |
| 系列预告 | 长节目和频道内容 | 引导回原视频的开放悬念 |

## 剪辑流程

1. 梳理原视频：标题、时长、章节、主题和观众。
2. 找候选片段：看开头强度、独立上下文和平台适配度。
3. 改写开头：从结果、冲突、问题或视觉反差开始。
4. 压缩上下文：只保留理解片段所需的信息。
5. 添加字幕和视觉强调。
6. 生成 Shorts、Reels、TikTok 和广告素材变体。
7. 检查节奏、清晰度和下一步动作。
